# LEARNINGS.md

> 长期经验和重复陷阱 — Agent 自动追加，不需要手动维护
> 格式：`## YYYY-MM-DD | 标题` → 症状 → 原因 → 解决

---

## 使用说明

- 每次踩坑或发现可复现的经验时，追加一条记录
- 用三段式：**症状** → **原因** → **解决**
- 做相似任务前先扫一遍相关条目
- 不追求完美格式，重点是能快速检索到
